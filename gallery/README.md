# Gallery

Annotated exemplars. **Inspiration, not menu.**

> **See the visual archive:** open [`index.html`](./index.html) in a browser — a Saville-mode editorial gallery that frames the annotations below.

## How to use

**Do:**
- Study the annotations — why does each work?
- Use the analysis to inform **new** compositions
- Read across multiple exemplars before generating; look for the pattern beneath the patterns

**Don't:**
- Copy layouts 1:1
- Use exemplars as templates
- Generate outputs that "look like" these

## The four-axis analysis

Every exemplar is analyzed along four axes:

1. **Palette** — where the colors come from, why they work
2. **Layout** — composition logic, grid decisions, violations
3. **Motion** — which motion categories, how they compose
4. **Type** — face pairings, scale decisions, voice

The generator's quality gate compares new outputs against these axes and rejects if similarity to any recent gallery output exceeds 0.7.

## Contributing exemplars

When adding an exemplar, include:
- Source (URL, author, year)
- The four-axis analysis
- "What to carry forward" — principles, not patterns
- "What to NOT carry forward" — elements that risk staling if reused
- Persona attribution — which persona's method does this exemplify?

The gallery grows. That's the point. What must NOT grow is the expectation that outputs resemble what's already here.
