Controlled Chaos PRO

A design system skill that generates novel web output.

Principles-based composition. Authorial personas. Subject-derived aesthetics. Open source. MIT licensed. No menu.

How it differs from v1

The original Controlled Chaos skill ships 5 aesthetic modes, 8 mood palettes, and 5 typographic voices. Two-hundred combinations. After six months of heavy use, every output becomes recognizable as “CC.” Pattern fatigue.

PRO removes the menu. Aesthetic DNA is derived per-run from the subject. Method comes from a persona library. Every output is one of one.

cd ~/.claude/skills

Axioms

  1. Type is the hero.
  2. Tension is beauty.
  3. Whitespace is pressurized.
  4. Motion has meaning.
  5. Derive, don’t decorate.

This page is a Rams-mode validation. It looks this quiet because the method demanded it — not because restraint is the aesthetic. The louder this page got, the less honest it became.
